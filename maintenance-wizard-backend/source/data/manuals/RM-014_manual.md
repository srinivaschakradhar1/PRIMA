# Equipment Manual: Coil Forming System (RM-014)

**Category:** Rolling Mills - Wire Rod Mill
**Equipment Type:** mechanical/rotating
**Document No:** MAN-RM-014-01
**Revision:** 1.0

## 1. General Description

The Coil Forming System (RM-014) is a key asset within the Rolling Mills - Wire Rod Mill of the integrated steel plant.
It is classified under equipment type "mechanical/rotating" and is monitored using a set of process and
condition-monitoring sensors as described in Section 3.

## 4. Operating Principle

The equipment operates as part of the continuous/batch process flow of the Rolling Mills - Wire Rod Mill.
Operating parameters should be maintained within the ranges specified in Section 3. Any sustained
deviation outside normal range should be investigated per the associated Maintenance SOP
(document SOP-RM-014-01).

## 6. Safety Precautions

1. Always follow Lock-Out/Tag-Out (LOTO) procedures before any maintenance activity.
2. Use appropriate PPE (helmet, safety shoes, gloves, ear protection, heat-resistant gear where applicable).
3. Ensure equipment is isolated from electrical, hydraulic, and pneumatic energy sources prior to intervention.
4. Confirm equipment has cooled to safe handling temperature where thermal hazards exist.
5. Obtain work permit / hot work permit as applicable before commencing work.

## 7. Maintenance Philosophy

This equipment is maintained under a combination of:
- Preventive Maintenance (time-based, per SOP-RM-014-01)
- Predictive/Condition-Based Maintenance (using sensor data summarized in Section 3)
- Corrective Maintenance (in response to fault alerts / breakdowns)
